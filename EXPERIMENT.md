# Experimental Falsification Protocol

This model lives or dies by **ΔF_anom**, the anomalous force residual
measured *after* all standard calibrations.

---

## Experimental Setup (Reference)

- Geometry: sphere–plane (PFA)
- Sphere radius: **R = 5 μm**
- Separation: **d = 100 nm**
- Baseline Casimir force:  
  **F_std ≈ 104 pN**
- Observable:  
  **ΔF_anom = F_measured − F_calibrated**

4. **Three-person tests**
